# Examples Folder — Ukázkové projekty pro testování proxy

## Popis

Přidání složky `examples/` s malými ukázkovými projekty (Python/uv, npm, .NET Core), které demonstrují použití Shieldoo Gate jako lokální proxy pro správu závislostí.

### Proč

- Uživatelé potřebují rychlý způsob, jak vyzkoušet proxy bez vlastního projektu
- Slouží jako smoke test — ověření, že proxy správně zprostředkuje instalaci balíčků
- Dokumentační hodnota — ukazují konfiguraci klientů proti lokální instanci
- Každý projekt má jednu závislost (+ tranzitivní), takže test je rychlý a deterministický

## Aktuální stav

Složka `examples/` neexistuje. Porty proxy jsou definovány v `config.example.yaml:6-11`:

| Protokol | Port | Upstream |
|----------|------|----------|
| PyPI | 5000 | `https://pypi.org` |
| npm | 4873 | `https://registry.npmjs.org` |
| NuGet | 5001 | `https://api.nuget.org` |
| Docker | 5002 | `https://registry-1.docker.io` |

Docker example je out of scope — vyžaduje konfiguraci Docker daemonu, což je výrazně složitější než klientská konfigurace pip/npm/dotnet.

### Konfigurace

Každý ukázkový projekt má vlastní konfiguraci klienta proti localhost:

- **Python**: `--index-url http://localhost:5000/simple/` předávaný jako CLI parametr pro `uv pip install`
- **npm**: `.npmrc` s `registry=http://localhost:4873/` (scope pouze na adresář projektu)
- **NuGet**: `nuget.config` s lokálním zdrojem na `http://localhost:5001/v3/index.json`

## Rizika a mitigace

| Riziko | Dopad | Pravděpodobnost | Mitigace |
|--------|-------|-----------------|----------|
| Pinnuté verze závislostí zastarají | Nízký | Střední | Verze jsou v README, snadno aktualizovatelné |
| Uživatel nemá nainstalované uv/node/dotnet | Nízký | Střední | README s prerekvizitami + každý příklad je nezávislý |
| Port 5000 konflikt s AirPlay na macOS | Nízký | Střední | Zmínka v examples/README.md s workaround |
| Upstream nedostupný (air-gapped prostředí) | Střední | Nízký | README prerekvizita: proxy musí mít přístup k upstream registries |

### Integrační / manuální testy

1. Spustit Shieldoo Gate via `docker compose -f docker/docker-compose.yml up`
2. Ověřit health: `curl http://localhost:8080/api/v1/health`
3. V `examples/python-requests/`: `uv pip install --index-url http://localhost:5000/simple/ -r requirements.txt && python main.py`
4. V `examples/npm-chalk/`: `npm install && node index.mjs`
5. V `examples/dotnet-json/`: `dotnet restore && dotnet run`
6. Ověřit v admin UI (localhost:8080), že se artefakty objevily

### Verifikace

```bash
# Ověřit, že všechny soubory existují
ls examples/python-requests/main.py examples/npm-chalk/index.mjs examples/dotnet-json/Program.cs

# Ověřit, že jsou porty správné
grep -r "localhost:5000" examples/
grep -r "localhost:4873" examples/
grep -r "localhost:5001" examples/

# Ověřit exact pinning (žádné ^ nebo ~ v package.json)
grep -E '[\^~]' examples/npm-chalk/package.json  # měl by být prázdný výstup
```

## Poznámky

- Příklady jsou záměrně minimální — cílem není demonstrovat kompletní projekt, ale ukázat konfiguraci proxy
- Každý příklad je zcela nezávislý, lze je spouštět jednotlivě
- Verze závislostí jsou pinnuté pro reprodukovatelnost
- `requests` stahuje tranzitivní závislosti (urllib3, certifi, idna, charset-normalizer) — to je záměr, ukazuje proxy v akci s více artefakty
- `.npmrc` scope je lokální na adresář — neovlivňuje globální npm konfiguraci
